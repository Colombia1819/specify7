The Code:

The code is available on github here. There are instructions there on how to run the demo locally. Of primary interest is the jQuery front end, which at this point consists of three files:

  • specifyform.js processes existing Specify resources to build DOMs in the browser for displaying Specify views
  • populateform.js pulls data from the RESTful back end and uses the functionality in specifyform.js to build and populate a view for a particular object.
  • putform.js extracts data from a view and updates the back end via HTTP PUT.
